## Why Freelancer Players Use It

- **Vanilla and mod-aware:** resolves names, systems, bases, ships, factions, equipment, cargo, and map data from the selected Freelancer install or mod folder.
- **Safer savegame writing:** validates risky changes, creates backup snapshots, and keeps restore options close at hand.
- **Encrypted save support:** handles Freelancer `FLS1` savegames so edited saves can remain compatible with the original game format.
- **Visual editing:** change reputation, Trent appearance, ship gear, cargo, visited map state, and jump connections through an editor instead of raw save rows.
- **Built for modded Freelancer:** useful for Freelancer HD Edition, total conversions, custom equipment packs, adjusted systems, and mod-specific save roots.
